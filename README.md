# Översättningsapp för Skolan

En översättningsapp för lärare som automatiskt översätter svensk text till spanska och ukrainska, med möjlighet att välja ytterligare språk.

## Språk som stöds

**Alltid inkluderade:**
- 🇪🇸 Spanska (latinamerikansk)
- 🇺🇦 Ukrainska

**Valfria språk:**
- 🇵🇰 Urdu
- 🇬🇧 Engelska
- 🇬🇷 Grekiska
- 🇹🇷 Turkiska
- 🇪🇷 Tigrinja
- 🇰🇪 Swahili (kenyansk)
- 🇧🇩 Bangla
- 🇸🇦 Arabiska (Modern Standard)
- 🇷🇺 Ryska
- 🇨🇳 Kinesiska (Mandarin, förenklad)
- 🇹🇭 Thailändska
- 🇲🇳 Mongoliska
- 🇦🇫 Dari
- 🇮🇷 Persiska

## Installation på GitHub Pages

1. Skapa ett nytt repository på GitHub
2. Ladda upp filen `index.html` till repot
3. Gå till Settings → Pages
4. Under "Source", välj "main" branch
5. Klicka "Save"
6. Din app kommer vara tillgänglig på: `https://dittanvändarnamn.github.io/reponamn`

## Användning

1. Skriv eller klistra in svensk text i textrutan
2. Välj eventuella extra språk utöver spanska och ukrainska
3. Klicka "Översätt"
4. Markera och kopiera översättningarna du behöver

## Teknisk information

Appen använder Claude AI (Anthropic) för översättningar, vilket ger betydligt bättre kvalitet än standardöversättningstjänster. Översättningarna är anpassade för skolkontext och pedagogiskt lämpligt språk.
